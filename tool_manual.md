## 🚀 工具简介

**DockerSystemdManager** 让您的 Docker 容器随系统自启动，轻松管理容器服务。

**核心功能：**
- 🔄 容器自启动
- 📊 服务生命周期管理  
- 📝 日志查看
- 🛡️ 自动故障重启
